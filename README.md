# ⚡ LangChain Summarizer - YouTube & Website

A Streamlit web application that summarizes content from **YouTube videos** and **any website** using **LangChain**, **Groq** (Llama 3.1), and advanced document loaders.

---

## ✨ Features

- **YouTube Video Summarization** (with transcript extraction)
- **Website Article Summarization**
- Powered by **qwen/qwen3-32b** via Groq (fast & efficient)
- Clean and user-friendly interface
- Error handling and input validation

---

## 🛠️ Technologies Used

- **Streamlit** - Web framework
- **LangChain** - LLM orchestration & chains
- **Groq** - Fast inference with Llama 3.1
- **UnstructuredURLLoader** - Website content extraction
